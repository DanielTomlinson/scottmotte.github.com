SendGrid Node.js url

2013

SendGrid is an email delivery API. They have a node.js library to interact with the API.

One of the first things I did when joining SendGrid was clean up this node.js library. Most significantly, I converted it to raise errors in a proper node-like fashion.

I continue to maintain the library and mostly manage pull requests from other contributors.
