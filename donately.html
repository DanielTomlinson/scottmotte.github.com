Donately url

2012

Donately is a software platform for accepting donations. It is owned and operated by Fifty & Fifty.

I was brought on to evaluate Donately and determine what direction it should take next. With my feedback, we decided to build out an API. Once completed we would design a new interface to consume that API.

We did just that. I directed and did the majority of the programming, plus added additional features, and maintained the old site for current clients.

Technologies include:

  • Ruby on Rails
  • RESTful API
  • Angular.js
  • PostgreSQL
  • Authorize.net
  • Stripe
  • PayPal
  • Heroku
